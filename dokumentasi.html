Instalasi

Panduan instalasi pustaka memories

Menggunakan CDN 

<script src="https://cdn.jsdelivr.net/npm/@wadahkode/memories@1.1.81/build/memories.min.js"></script>

Tidak ada konfigurasi jika anda menggunakan CDN, cukup buat element baru seperti dibawah ini: 

<time datetime="12/7/2020, 16:11:00" prefix="birthday|expired|schedule|countdown"></time>

Datetime attributes:

  1. 1. datetime
    Isi dengan tanggal, rekomendasi gunakan format seperti ini: 
    datetime="12/7/2020, 16:11:00"
    Penjelasan:
    • - Angka 12 adalah bulan
    • - Angka 7 adalah hari
    • - Angka 2020 adalah tahun
    • - Angka 16 adalah jam
    • - Angka 11 adalah menit
    • - Angka 00 adalah detik
  2. 2. prefix
    Isi dengan birthday atau expired atau schedule atau countdown, contoh: 
    prefix="birthday"

  3. Modifikasi dengan konfigurasi

    Secara otomatis pustaka memories akan mengembalikan sebuah object baru {ref: f}, yang dapat anda gunakan di browser untuk melakukan konfigurasi secara manual dari mulai mengatur bahasa, menentukan tanggal, menentukan zona waktu dan lain-lain.

    Untuk lebih lanjut lihat contoh dibawah ini: 

    <script>
  const ref = Memories.ref();
  ref.set('datetime', '1/1/1970');
  // Cetak umur
  console.log(ref.timeAgo('birthday'))
</script>

Menggunakan NPM

Pustaka memories telah tersedia di NPM, untuk mengunduhnya anda dapat menggunakan perintah dibawah ini: 

npm install @wadahkode/memories --save-dev

Cara menggunakan pada lingkungan nodejs 

const memories = require('@wadahkode/memories')
const moment = new memories();
// atur tanggal
moment.set('datetime', '1/1/1970');
// cetak umur
console.log(moment.timeAgo('birthday'))

Untuk lebih jelasnya anda dapat membaca dari sini README.md

Tentang memories

Memories adalah sebuah library javascript yang digunakan untuk mengubah format waktu menjadi sebuah momen seperti waktu lalu, kemudian menentukan expired dari sebuah produk, membuat jadwal dan membuat hitungan mundur.

Sejarah memories

Berawal dari ketidaktahuan saya terhadap sebuah library javascript yaitu momentjs, saya membuat sendiri penghandle waktu untuk ditampilkan menjadi baru saja, beberapa detik yang lalu, berapa menit yang lalu, berapa jam yang lalu, berapa hari yang lalu, minggu, bulan dan tahun.

Awalnya memories hanya mendukung untuk memformat seperti diatas, tetapi ketika ada seseorang yang bertanya pada sebuah group telegram, akhirnya saya mengembangkan untuk mendukung lebih banyak format waktu. dan berikut beberapa dukungan yang saat ini tersedia:
  • 1. Timeago (Format waktu yang berlalu)
  • 2. Expired (Menentukan kadaluarsa dari sebuah produk)
  • 3. Schedule (Membuat sebuah jadwal waktu)
  • 4. Countdown (Menghitung waktu mundur)
  • 5. Birthday (Format waktu untuk umur)

Menggunakan NPM

Menggunakan pustaka memories di lingkungan nodejs memang lebih fleksibel, karena kita dapat mengatur apa saja yang diperlukan, oke mari koding: 
const memories = require('@wadahkode/memories);
const moment = new memories();

moment.set('datetime', '1/1/1970, 00:00:00');
moment.set('language', 'id-ID');
moment.set('locale', {timeZone: 'Asia/Jakarta', hour12: false});

// waktu lalu tanpa parameter
moment.timeAgo();
// waktu lalu dengan kustom keluaran
moment.timeAgo('timeAgo', {
y: 'tahun yang lalu',
m: 'bulan yang lalu',
d: 'hari yang lalu',
h: 'jam yang lalu',
i: 'menit yang lalu',
s: 'detik yang lalu',
n: 'baru saja'
});
// mencetak umur
moment.timeAgo('birthday');
